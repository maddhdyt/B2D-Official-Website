# Globe Texture Credits

- `earth-night.webp` is derived from NASA Earth Observatory's Black Marble 2016 color map.
- `earth-clouds.webp` and `starfield.webp` are generated procedurally by `scripts/generate-globe-assets.mjs`.

NASA source imagery:

- https://science.nasa.gov/earth/earth-observatory/earth-at-night/maps/

## Portfolio Showcase

The files in `portfolio/` were generated with the built-in OpenAI image
generation tool, then resized to 1200x800 and encoded as AVIF/WebP with Sharp.

- `analytics.*`: futuristic marketing analytics and performance dashboard.
- `creative.*`: premium branding and UI/UX design workspace.
- `crm.*`: enterprise CRM, lead nurturing, and automation platform.

All prompts requested dark, cinematic agency case-study covers without logos,
watermarks, people, or readable brand names.
